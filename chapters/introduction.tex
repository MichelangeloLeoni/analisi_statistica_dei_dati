\chapter{Introduction}

These notes are intended for a course in statistical data analysis. They provide theoretical foundations and practical tools for understanding data.

\begin{definition}
A random variable is a measurable function
\[
X : \Omega \rightarrow \R
\]
\end{definition}

\begin{example}
If $X \sim \N(0,1)$, then
\[
\E[X] = 0
\]
\end{example}
