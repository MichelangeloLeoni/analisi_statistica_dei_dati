\section{Introduzione}

Queste sono dispense per il corso di analisi statistica dei dati.

\begin{definizione}
Una variabile casuale è una funzione misurabile
\[
X : \Omega \rightarrow \R
\]
\end{definizione}

\begin{esempio}
Se $X \sim \N(0,1)$ allora
\[
\E[X] = 0
\]
\end{esempio}
